
## Overview

This little project aims to solve Flow Free puzzles via my DlxLib.

## Example Puzzle

Example 7x7 puzzle:

```
------A
-----BC
-B-----
---DE--
--D-F--
----CF-
-----AE
```

And its solution:

```
AAAAAAA
ABBBBBC
ABCCCCC
ACCDEEE
ACDDFFE
ACCCCFE
AAAAAAE
```

Each letter represents a colour

* A: Blue
* B: Orange
* C: Red
* D: Green
* E: Turquoise
* F: Yellow
